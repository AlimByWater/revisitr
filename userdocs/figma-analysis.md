## 2. Дизайн-система

### Визуальный стиль
- **Палитра**: минималистичный чёрно-белый с текстурным фоном (noise grain)
- **Акцент**: красный/оранжевый — логотип «rev/**s**itr», PRO-бейдж, ссылки
- **Шрифт**: sans-serif, чистая иерархия размеров
- **Уровень проработки**: wireframe / low-fidelity — структура и навигация без финального визуала

## 5. Заметки для разработки

1. **Дизайн на уровне wireframe** — при реализации потребуется UI-библиотека для финального вида (shadcn/ui, Ant Design, или кастомная)
2. **Sidebar — ключевой компонент** — collapsible, с вложенностью до 3 уровней, контекстно меняется (например, показывает список ботов)
3. **Модальные окна** — используются для создания сущностей (бот, верификация)
4. **Статусы** — двухуровневая система: «активен» (зелёный) / «требуется действие» (жёлтый)
5. **Текстурный фон (noise)** — уникальный визуальный элемент бренда, нужно сохранить
6. **«Маркетинг под ключ»** — отдельный пункт в header, возможно upsell-сервис
